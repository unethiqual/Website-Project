Название проекта: веб-приложение для управления личными финансами «SuperFinance»

Автор проекта: Демидов Арсений Михайлович

Описание идеи: Идея заключается в создании удобного многофункционального веб-приложения для администрации личных финансов различными пользователями, получить доступ к которому можно с любого устройства.

Описание реализации:
А) Основные файлы и директории:
	1) папка /app/db - хранение логики подключения к БД
	2) папка css - хранение CSS файлов
	3) папка errors_mess - хранение фидбека и сообщений об ошибках от пользователей
	4) папка logs - хранение сервеных логов
	5) папка templates - хранение HTML-шаблонов страниц
	6) config.py и .env - конфигурационные файлы
	7) mail.py - логика отправки электронной почты
	8) main.py - основная логика приложения 
Б) Технологии проекта
	1) Flask — основной веб-фреймворк для построения бэкенда и маршрутизации.
	2) Werkzeug — используется для логирования и работы с HTTP сервером.
	3) Matplotlib (plt) — используется для построения графиков и визуализации.
	4) Flask-SQLAlchemy - используется для работы с БД
	5) logging - используется для логирования
	6) smtplib - для отправки электронных писем

В) Основные эндпоинты
1. index
    - Главная страница сайта.
    - Обычно возвращает приветствие или стартовую информацию.

2. login_signup
    - Обработка логина и регистрации пользователей.
    - Осуществляет вход или создание новой учетной записи по переданным данным.

3. check_email
    - Проверяет корректность или существование email, возможно, для регистрации или восстановления доступа.

4. manage_expenses
    - Управление расходами пользователя.
    - Добавление, редактирование, удаление или просмотр расходов.

5. check_balance
    - Проверка текущего баланса пользователя на основе введённых доходов и расходов.

6. settings
    - Страница или API для изменения пользовательских настроек.

7. feedback
    - Приём и обработка обратной связи от пользователя, а также сообщений об ошибках сервиса.

8. show_plots
    - Генерация и отображение графиков (визуализация расходов, доходов и других финансовых данных пользователя).

9. manage_incomes
    - Управление доходами пользователя.
    - Аналогично расходам: добавление, редактирование, удаление и просмотр.

10. logout
    - Выход пользователя из системы (завершение сессии).

Необходимые для установки файлы приведены в файде requirements.txt, который находится в корневой директории проекта
